# 🌐 File Wiki: nginx.conf & kong.yml

## Overview
`nginx.conf` implements hardened HTTP response headers (Content-Security-Policy, HSTS, X-Frame-Options) and TLS 1.3 encryption. `kong.yml` configures declarative Zero-Trust API gateway plugins, rate-limiting, and JWT authentication.
